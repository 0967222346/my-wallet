# もにゃ

初心者でも簡単に使えるように

* 技術的要素をできるだけ排除し
* 難しい専門用語でなくわかりやすい言葉で
* 暗号通貨を使う上での豆知識の教本として使え
* 上級者でも満足できるように設定できる

モナコインウォレット
